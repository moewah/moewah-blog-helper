# 文章分类定义

## 分类列表

| 分类 | 适用内容 |
|------|---------|
| Astro教程 | Astro 框架使用、组件开发、部署配置 |
| AI实验室 | AI 工具使用、Prompt 工程、模型部署 |
| NAS私有云 | NAS 硬件、存储方案、数据管理 |
| 私有化部署 | 自托管服务、Docker 部署、服务器配置 |
| 虚拟化与运维 | 虚拟机、容器、DevOps、监控告警 |
| 网络与安全 | 网络配置、安全防护、VPN、证书 |
| 硬件教程 | 硬件评测、装机、外设使用 |
| 增长与SEO | SEO 优化、流量增长、内容运营 |
| 认知与成长 | 个人成长、方法论、读书笔记 |
| 光影与生活 | 摄影、旅行、生活记录 |
| 未分类 | 无法归类的文章 |

## 分类决策规则

1. 根据**文章主体**和**最终交付物**判断
2. 教读者**做一件事** → 技术/工具类
3. 启发读者**思考** → 内容/认知类
4. 模糊时选择最接近的分类
